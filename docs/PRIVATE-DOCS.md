# Private Docs Convention

Personal notes and non-public material should live in `docs/private/`, which is gitignored.

Suggested files:

1. `docs/private/PROJECT_STATUS.md`
2. `docs/private/09-PROGRESS-LOG.md`
3. `docs/private/SESSION_HANDOFF_TEMPLATE.md`
4. `docs/private/11-TECHNICAL-OUTCOMES.md`
5. `docs/private/12-ARCHITECTURE-QA.md`

Do not commit files from `docs/private/` unless you intentionally remove that ignore rule.
